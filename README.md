# AI-Powered-Robo-Advisor

![CI](https://github.com/marcovernocchi/AI-Powered-Robo-Advisor/actions/workflows/ci.yml/badge.svg)

A personalized investment advisory platform combining machine learning portfolio optimization, real-time market data, and AI-generated financial advice via an LLM. Built as a university project for Programming in Finance II (2026).

## Features

- **User authentication** — secure JWT-based registration and login
- **Risk profiling** — 4-section questionnaire (MiFID II-style) that calculates a score from 8 to 68, with a plain-language AI explanation of the result
- **Multi-portfolio management** — create multiple portfolios, view an aggregated net-worth dashboard, and toggle which portfolios count towards it
- **Real-time holdings** — live prices and P&L from Yahoo Finance, with per-holding purchase date, fees, notes and asset type
- **Multi-currency support** — each holding is tracked in its native currency and converted to your chosen display currency via live FX rates
- **CSV / Excel import** — upload a broker export; columns and asset types are detected automatically (AI-assisted via Groq, with a rule-based fallback)
- **Excel / PDF export** — download a formatted portfolio report with holdings, allocation chart, optimizer weights and historical performance metrics
- **Portfolio optimization** — Black-Litterman (market-cap prior) for the aggregated portfolio, classic mean-variance for single portfolios, plus a Hierarchical Risk Parity (HRP) model, all tuned to your risk profile
- **Model portfolio suggestions** — a ready-made ETF allocation per risk band (Defensive / Conservative / Balanced / Aggressive) with an AI-generated explanation, shown to users before they add holdings
- **AI advice** — personalized investment advice generated by Llama 3.3-70B (Groq, free tier)
- **Backtesting** — simulate a portfolio's historical performance with configurable rebalancing, transaction costs, TER, spread, and benchmark comparison (return, CAGR, Sharpe/Sortino, drawdown, VaR/CVaR, rolling metrics)
- **Monte Carlo simulation** — projects a range of future portfolio outcomes from historical return/volatility estimates (via the `/monte-carlo` API)
- **Market explorer** — price history charts, dividends, company info and ticker search/autocomplete for any stock, ETF or crypto
- **Multi-language UI** — English / Italian, switchable from Settings
- **React frontend** — modern dashboard built with React + Tremor (dark/light theme)
- **Streamlit frontend** — legacy prototype UI still available

## Project Criteria Satisfied

- **LLM** — Groq/Llama generates personalized financial advice, risk-profile explanations, portfolio suggestion rationale, and AI-assisted import parsing
- **ML models** — Mean-variance, Black-Litterman and Hierarchical Risk Parity (HRP) portfolio optimization; Monte Carlo simulation; historical backtesting with performance/risk metrics
- **Web frontend** — React dashboard with interactive charts, multi-portfolio views, and English/Italian i18n
- **Non-trivial database** — SQL with 6 related tables (users, portfolios, holdings, advice, price_cache, optimization_results)
- **Real-time data** — Live prices, dividends and FX rates via yfinance
- **Own API + auth** — FastAPI REST API with JWT authentication
- **Cloud infrastructure** — Deployed on DigitalOcean VPS

## Installation

### Prerequisites
- Python 3.11+
- Node.js 18+
- A [Groq API key](https://console.groq.com) (free)

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/marcovernocchi/AI-Powered-Robo-Advisor.git
cd AI-Powered-Robo-Advisor

# 2. Create and activate a virtual environment
python -m venv .venv
source .venv/bin/activate      # macOS/Linux
# .venv\Scripts\activate       # Windows

# 3. Install Python dependencies
pip install -r requirements.txt

# 4. Set up environment variables
cp .env.example .env
# Edit .env and add your GROQ_API_KEY and a SECRET_KEY

# 5. Start the backend (terminal 1)
uvicorn backend.main:app --reload

# 6. Install and start the React frontend (terminal 2)
cd frontend-react
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

The FastAPI interactive docs are at [http://localhost:8000/docs](http://localhost:8000/docs).

### Legacy Streamlit frontend (optional)

```bash
PYTHONPATH=. streamlit run frontend/app.py
# Available at http://localhost:8501
```

## Usage

1. **Register** an account on the login page (your country sets your default display currency)
2. **Complete the risk questionnaire** in the AI Advisor page (4 sections, MiFID II style) and read the AI-generated explanation of your profile
3. **Review the suggested model portfolio** for your risk band, or **add holdings** in the Portfolio page (manually or by importing a CSV/Excel broker export)
4. **Create additional portfolios** if needed — the Dashboard aggregates all portfolios marked "include in net worth"
5. **Run optimization** to see Black-Litterman / mean-variance / HRP weights for your risk profile
6. **Generate AI advice** for personalized commentary on your portfolio
7. **Backtest** a target allocation with custom rebalancing, costs and a benchmark on the Backtesting page
8. **Export** your portfolio as Excel or PDF from the Portfolio page
9. **Explore** any stock, ETF or crypto in the Market page
10. **Adjust** country, display currency and language in Settings

## API Documentation

The full interactive API is available at `/docs` when the backend is running.

**Auth**

| Method | Endpoint | Description |
|---|---|---|
| POST | `/auth/register` | Create account (country sets the default display currency) |
| POST | `/auth/login` | Get JWT token |
| GET | `/auth/me` | Current user info |
| PATCH | `/auth/me` | Update country / display currency |

**Risk profile**

| Method | Endpoint | Description |
|---|---|---|
| POST | `/risk-profile` | Submit risk questionnaire, get score (8–68) and profile bands |
| POST | `/risk-profile/explain` | AI-generated plain-language explanation of the profile |

**Portfolio**

| Method | Endpoint | Description |
|---|---|---|
| GET | `/portfolio/` | Aggregated holdings (across portfolios marked "include in net worth") with live prices, converted to display currency |
| GET | `/portfolio/list` | List the user's portfolios with totals |
| POST | `/portfolio/create` | Create a new portfolio |
| GET | `/portfolio/{id}` | Get a single portfolio with its holdings |
| PATCH | `/portfolio/{id}` | Rename a portfolio / toggle "include in net worth" |
| DELETE | `/portfolio/{id}` | Delete a portfolio |
| POST | `/portfolio/holdings` | Add a holding (ticker, shares, avg price, currency, asset type, purchase date, fees, notes) |
| PATCH | `/portfolio/holdings/{id}` | Update a holding (e.g. record a partial sale) |
| DELETE | `/portfolio/holdings/{id}` | Remove a holding |
| GET | `/portfolio/optimize` | Optimize the aggregated portfolio (Black-Litterman), persists the result |
| GET | `/portfolio/optimize/{id}` | Optimize a single portfolio (mean-variance) |
| GET | `/portfolio/metrics` | Return, volatility, equity share and diversification of the aggregated portfolio |
| GET | `/portfolio/suggestions` | AI-explained model portfolio for the user's risk band |
| POST | `/portfolio/import/preview` | Parse a CSV/Excel broker export and preview detected holdings |
| POST | `/portfolio/import/confirm` | Import the previewed holdings into a portfolio |
| GET | `/portfolio/export/excel` / `/portfolio/{id}/export/excel` | Download a portfolio report as Excel |
| GET | `/portfolio/export/pdf` / `/portfolio/{id}/export/pdf` | Download a portfolio report as PDF |

**Market**

| Method | Endpoint | Description |
|---|---|---|
| GET | `/market/price/{ticker}` | Current price |
| GET | `/market/history/{ticker}` | Price history (period or start date) |
| GET | `/market/dividends/{ticker}` | Dividend history |
| GET | `/market/info/{ticker}` | Company / asset info |
| GET | `/market/search?q=` | Ticker search/autocomplete |

**Advice**

| Method | Endpoint | Description |
|---|---|---|
| POST | `/advice/generate` | Generate AI advice for the current portfolio |
| GET | `/advice/history` | Past advice (last 5) |

**Backtesting & simulation**

| Method | Endpoint | Description |
|---|---|---|
| POST | `/backtest` | Run a historical backtest (rebalancing, costs, benchmark, full metrics) |
| POST | `/monte-carlo` | Run a Monte Carlo projection of portfolio value |

## Deployment on DigitalOcean

1. Create a **Droplet** (1 vCPU, 1 GB RAM) on DigitalOcean
2. Install Docker on the Droplet
3. Clone the repo, create `.env`, then:
   ```bash
   docker compose up -d
   ```
4. Open ports 8000 (API) and 3000 (React frontend) in the DigitalOcean firewall settings
